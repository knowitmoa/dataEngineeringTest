# dataEngineeringTest

1. Install docker

1. Clone the repository:

   ```bash
   git clone https://github.com/knowitmoa/dataEngineeringTest.git

   cd repo
   ```

1. Run in terminal

```bash
  docker-compose up -d
```

3. Install dependencies:
   ```bashs
   pip install -r requirements.txt
   ```
4. Start program:
   ```bash
   python start.py
   ```
